Sistema de login para cadastro de alunos e anotações como regras e pagamento da mensalidade de forma simples como anotação.
O cadastro ocorre soomente pelo administrador que libera o acesso ao aluno para acessar, editar a senha e começar a ler as apostilas e vídeos aulas.
O sistema foi criado como experimento de estudos e habilidades com grande interesse em banco de dados quando eu dava aula de defesa pessoa.
